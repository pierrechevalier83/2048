2048 - game
-----------
A simple C++/ncurses implementation of the popular game

Download
--------
git clone --recursive git@github.com:pierrechevalier83/2048.git

Build
-----
- `mkdir build`
- `cd build`
- `cmake ..`
- `make`

Run
---
- `./2048`

Install
---
- `sudo make install`
