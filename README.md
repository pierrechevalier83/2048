2048 - game
-----------
A simple C++/ncurses implementation of the popular game
![alt tag](https://raw.githubusercontent.com/pierrechevalier83/2048/master/demo.gif)

Download
--------
`git clone --recursive git@github.com:pierrechevalier83/2048.git`

Build
-----
- `mkdir build`
- `cd build`
- `cmake ..`
- `make`

Run
---
`./2048`

Install
---
`sudo make install`
